# Search-Based Software Engineering (SBSE)

This repository contains my coursework and projects for the Search-Based Software Engineering (SBSE) course for the Winter Semester 24/25.

## Projects Included

- **Test Prioritisation** (`ws2425sbsetesprio-dagogo01`): Implementation of various search algorithms (Genetic Algorithms, Simulated Annealing, Random Search) for test case prioritisation.
- **Suite Minimisation** (`ws2425sbsesuiteminimisation-dagogo01`): Multi-objective search algorithms (e.g., NSGA-II) applied to test suite minimisation.
- **NEAT** (`ws2425sbseneat-dagogo01`): NeuroEvolution of Augmenting Topologies implementation for SBSE tasks.
- **Hill Climbing** (`ws2425sbsehillclimbing-dagogo01`): Basic implementations of hill climbing algorithms for software engineering problems.

## Structure
Each project is a Maven-based Java application. You can find the source code within the respective `src/` directories and tests within the `test/` directories.
